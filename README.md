## 🚀 快速部署 (Quick Start)

这是一个纯静态的高级网页项目，支持秒级一键免服务器部署：

1. **Fork 本仓库** 到您自己的账户下。
2. 进入仓库设置：`Settings` -> `Pages`。
3. 将 **Build and deployment** 下的 Source 设置为 `Deploy from a branch`。
4. Branch 选择 `main` 分支（或 `master`），目录选择 `/root`，点击 **Save**。
5. 刷新页面，即可获得您专属的二级域名官方站点！
